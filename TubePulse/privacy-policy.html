TubePulse Privacy Policy

Last updated: February 23, 2026

Overview

TubePulse is a Chrome extension that analyzes YouTube video comments to provide content improvement suggestions. We are committed to protecting your privacy.

Data Collection

TubePulse collects the following data:

  • Anonymous User ID: A random UUID is generated locally and stored on your device to track monthly usage quotas. This ID is not linked to any personal information.
  • Email (optional): If you sign up for the Pro waitlist, your email address is sent to and stored on our server. It is used solely to notify you when Pro launches.
  • We do not use cookies or tracking technologies
  • We do not have user accounts or registration

Data Processing

When you click "Analyze Comments", the following happens:

  • Basic video information (title, channel name, view count) is extracted from the page DOM
  • The video ID is sent to our API server, which fetches comments via the YouTube Data API
  • Comments and video metadata are forwarded to the DeepSeek AI API for analysis
  • The AI-generated analysis results are displayed to you
  • Analysis snapshots (problem titles, counts — not full comments) are stored on our server to enable the Trend feature

Local Storage

TubePulse uses Chrome's local storage (chrome.storage.local) to save:

  • Your most recent analysis results (so they persist when you reopen the popup)
  • Your analysis history (up to 20 entries)

This data is stored entirely on your device and is never transmitted to any server. You can clear this data at any time using the "Clear All" button in the History section.

Third-Party Services

TubePulse uses the following third-party services:

  • YouTube Data API — to fetch video comments and metadata for analysis. Our use complies with the YouTube API Services Terms of Service. Also see Google's Privacy Policy.
  • DeepSeek API — for AI-powered comment analysis. Comment text and basic video information are sent to DeepSeek's servers for processing. Please refer to DeepSeek's Privacy Policy for their data handling practices.

Permissions

TubePulse requests the following Chrome permissions:

  • activeTab — to read basic video information (title, channel name, view count) from the current YouTube page when you trigger an analysis
  • storage — to save analysis history, preferences, and anonymous usage ID locally on your device
  • sidePanel — to display the analysis interface in Chrome's side panel alongside the YouTube video

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected on this page with an updated date.
